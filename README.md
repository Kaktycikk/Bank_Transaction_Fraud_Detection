# Bank Transaction Fraud Detection

## О проекте

В проекте проведен анализ данных и построены базовые модели машинного обучения для выявления мошеннических транзакций.

Цель — понять, какие признаки связаны с мошенничеством и насколько хорошо простые модели способны его обнаруживать.

---

## EDA

- Большинство числовых признаков (distance, city_pop) не показали значимой зависимости  
- Сумма транзакции (amount) имеет слабый сигнал — fraud чаще связан с большими значениями  
- Категория продавца (category) оказалась наиболее информативной  
- Сильных линейных зависимостей с целевой переменной не выявлено  

---

## Модели

Использованы:
- Logistic Regression  
- Random Forest  

Результат:
- модели плохо различают классы без балансировки  
- при `class_weight='balanced'`:
  - высокий recall (находит fraud)
  - низкий precision (много ложных срабатываний)

---

## Вывод

- В данных нет сильных однофакторных зависимостей  
- Мошенничество определяется комбинацией признаков  
- Качество моделей ограничено информативностью данных
